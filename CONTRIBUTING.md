# Contribuindo

Agradecemos seu interesse em contribuir com este projeto!

Este repositório faz parte de um Projeto Final de Curso (TFC), mas contribuições, sugestões e melhorias são bem-vindas.

---

## 🛠 Configuração de Desenvolvimento

1. Clone o repositório

git clone https://github.com/RafaelHDSV/Medflow.git

2. Instale as dependências

yarn

3. Execute o servidor de desenvolvimento

yarn dev

---

## 📂 Estrutura do Projeto

backend/ -> API Node.js
frontend/ -> Aplicativo React

---

## 🧾 Diretrizes de Codificação

- Siga os padrões de projeto existentes
- Escreva código claro e de fácil manutenção
- Use mensagens de commit descritivas
- Mantenha os pull requests pequenos e focados

---

## 🔀 Pull Requests

1. Faça um fork do repositório
2. Crie uma nova branch
3. Implemente suas alterações
4. Abra um Pull Request

Seu PR será revisado antes de ser mesclado.

---

## 🐛 Relatando Problemas

Use os modelos de issue do GitHub para relatar bugs ou solicitar recursos.

---

Obrigado pela sua contribuição!
